### 엘리먼트의 속성과 메서드

1. `textContent`
    - 엘리먼트의 텍스트를 저장
2. `innerHTML`
    - 엘리먼트 내부의 트리를 문자열 타입으로 저장
3. `classList`
    - 엘리먼트의 클래스 리스트를 저장
    - 클래스를 조작하기 위한 몇가지 메서드를 가지고 있다
    - `.add()` : 클래스 추가
    - `.remove()` : 클래스 삭제
    - `.contains()` : 특정 클래스를 포함하는지 체
4. `attribute`
5. `createElement`
